## 💻 Tech Stack
- Java 17, Spring Boot 3.2.1, Spring Data JPA
- AWS (EC2,CodeDeploy, RDS for MySQL)
- Docker Container
- Let's Encrypt for HTTPS
- Nginx for Web Server(Reverse-Proxy)
